## Current State

The codebase currently uses generic naming for Linux clipboard scripts:

- `linux_*` scripts use the `xclip` tool for X11 systems
- `wayland_*` scripts use `wl-copy`/`wl-paste` tools for Wayland systems

This naming is ambiguous because "linux" could refer to either X11 or Wayland backends.

## Naming Convention

After this change:

- **xclip\_\***: Scripts that use the `xclip` command (X11 backend)
- **wl*clipboard*\***: Scripts that use `wl-copy`/`wl-paste` commands (Wayland backend)
